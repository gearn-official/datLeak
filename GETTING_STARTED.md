## GETTING STARTED

### Prerequisites
- GCC installed
- Make installed
- Git installed

### Clone and build
```bash
git clone https://github.com/gearn-official/datLeak
cd datLeak/code
make
```

### Run a file
```bash
./datleak myfile.c
```

### Interactive mode
```bash
./datleak
```
Type your code and press Ctrl+D when done.
