# datLeak Changelog

## [0.1] - 2026-06-12

### Added
- Basic C interpreter/virtual machine
- printf with %d, %s, %c, %%, \n, \t
- int, char, long, short, unsigned variables
- Basic arithmetic: + - * /
- Operator precedence: * / before + -
- Unary negation: -x
- Parenthesized expressions: (x + y) * z
- Variable assignment: x = expr;
- return statement
- Single-line comments: //
- Block comments: /* */
- Preprocessor directives ignored: #include, #define
- File mode: ./datleak file.c
- Interactive mode: ./datleak
